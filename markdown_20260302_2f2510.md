# Paradise Nursery - Plant Shopping Application

A React-based e-commerce application for purchasing houseplants, featuring a shopping cart powered by Redux.

## Project Details

**Project Name:** Paradise Nursery  
**Description:** An online store for beautiful houseplants where users can browse different categories of plants, add them to cart, and manage their purchases.

## Features

- Browse houseplants by categories (Indoor Plants, Outdoor Plants, Succulents)
- Add products to shopping cart with disabled button after addition
- Dynamic cart icon showing total items count
- Shopping cart page with quantity management
- Total cost calculation per item and overall
- Responsive navbar with navigation links
- Persistent cart state using Redux

## Technologies Used

- React.js
- Redux Toolkit
- React Router DOM
- CSS3
